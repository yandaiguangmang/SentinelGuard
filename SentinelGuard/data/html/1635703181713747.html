Summary Report for:
27-3099.00 - Media and Communication Workers, All Other

All media and communication workers not listed separately.

The occupation code you requested, 27-3012.00 (Public Address System and Other Announcers), is no longer in use. In the future, please use 27-3099.00 (Media and Communication Workers, All Other) instead.

"All Other" titles represent occupations with a wide range of characteristics which do not fit into one of the detailed O*NET-SOC occupations. O*NET data is not available for this type of title.
