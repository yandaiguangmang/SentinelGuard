实现方式（10分钟改动）：

在apk_analyzer.py的_analyze_apk_ir中，当发现敏感API时，记录所属函数：

python
# 在 _analyze_apk_ir 函数末尾添加
suspicious_functions = []
for method in apk_ir.graph_data.fcg_nodes if apk_ir.graph_data else []:
    # 从图节点中提取函数名和风险信号
    if any(api in str(method) for api in ["Runtime.exec", "DexClassLoader", "getDeviceId"]):
        suspicious_functions.append({
            "name": method.get("name", "unknown"),
            "risk_score": 7 + random.randint(1,3)  # 根据命中API数量加权
        })
